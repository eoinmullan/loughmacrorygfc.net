# Loughmacrorygfc.net

Archive of the website for Loughmacrory GFC.

## How to Contribute

### 1. Get the Code

First, you'll need to clone (download) the website code to your computer. You'll need to install `git`, then:

```bash
git clone git@github.com:eoinmullan/loughmacrorygfc.net.git
cd loughmacrorygfc.net
```

### 2. Make Your Changes

- Edit the HTML files using any text editor
- Preview your changes by opening the `.html` files in your web browser
- Test that links and images still work

### 3. Submit Your Changes

Once you're happy with your changes, create a Pull Request:

1. Create a new branch: `git checkout -b your-branch-name`
2. Make your changes and commit them:
   ```bash
   git add .
   git commit -m "Brief description of what you changed"
   ```
3. Push your branch: `git push origin your-branch-name`
4. Go to GitHub and click "Create Pull Request"
5. Wait for review and approval

## Automatic Deployment

The site uses Cloudflare Pages for hosting. When changes are pushed to the `main` branch, the site automatically updates within a few minutes.
